# 项目简介

## 核心功能
- 系统首页展示
- 用户管理
- 知识产权人管理
- 产权类型管理
- 知识产权信息管理
- 知识产权变更管理
- 知识产权缴费管理
- 在线投诉处理
- 系统管理

## 适用场景
- 知识产权管理
- 企业知识产权信息化建设
- 政府知识产权管理部门
- 知识产权服务机构

## 技术特点
- 架构：B/S架构，MVC模式
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
